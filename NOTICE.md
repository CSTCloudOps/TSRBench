# Notice

This repository contains the release code for TSRBench.

The release keeps the UCR-R and CU-RCA benchmark data, lightweight paper
outputs, and experiment scripts in Git. Large pretrained model weights and
generated distance caches are intentionally excluded and should be downloaded
or regenerated locally.

Third-party components and model families used by the benchmark include
TS2Vec, CoST, TimesFM, Chronos, Timer, Sundial, Moirai, Time-MoE, TabPFN,
PyTorch, scikit-learn, SciPy, NumPy, pandas, and Hugging Face tooling. Please
check the corresponding upstream repositories and model cards for their own
licenses and usage terms before redistribution.
